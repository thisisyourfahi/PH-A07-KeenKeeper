 # KeenKeeper

 This project was completed as a task of Programming Hero. Recently I learned about Next.js, and i had to implement the concepts(contexts api, recharts) to buid this project. It was fun.

 ## Technology Used:
 * Next.js
 * React.js
 * Tailwindcss + daisyUI
 * Javascript(ES6+)
 * Recharts
 * React-Toastify(NPM Package)
 * JSON

 ## Features:
 * Functional Navbar
 * Functional Friends card
 * Functional action buttons(call, text, video)
 * Filterable timeline section
 * Dynamic friendship analytics section using Recharts
